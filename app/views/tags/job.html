Lancement de Job

On a déjà vu un exemple de Job lorsqu'on a évoqué l'insertion de données au démarrage de l'application. Cependant Play! propose d'étendre cette fonctionnalité de pouvoir créer des tâches planifiées très simplement. Nous avons crée un exemple de tâche planifiée définie dans FakeJob.java : 

/**
 * Sample of class to illustrate how Jobs work.
 */
// @Every("10s")
public class FakeJob extends Job≶Object> {

	@Override
	public void doJob() throws Exception {
		String log = MessageFormat.format(
				"Exemple de job effectué à : {0, date, hh:mm:ss}", new Date());
		Logger.info(log);
	}
}

Le simple fait d'ajouter l'annotation @Every("10s") et d'etendre la classe Job permet de créer une tâche planifiée, éxécutée en l'occurrence toutes les 10 secondes (c'est pourquoi l'annotation est commentée, cela évite de polluer vos log. Décommentez la ligne et relancé le serveur pour vérifier que ça marche)

Note : Pour des planifications plus compliquées, il est possible d'utiliser l'annotation @On("...") à la place de @Every("10s") avec en paramètre une expression CRON.
