Bienvenue sur le StarterKit Play!Framework Ausy

Si vous lisez cette page c'est que vous avez correctement installer Play!Framework et réussi à lancer le StarterKit Ausy
Ce StarterKit à bien sur pour objectif de vous aidez à commencer une application en utilisant Play en contenant déjà quelques classes et modules intéressants mais il contient également un support de formation se basant sur le mini-projet AusyPlay accessible sur github (git://github.com/kebuu/AusyPlay.git).

Si vous connaissez déjà bien Play!Framework et que le support de formation ne vous intéresse pas, vous pouvez supprimer tous ce qui concerne cette documentation et ne garder qu'un StarterKit vide et prêt à l'emploi. Dans le cas contraire il est conseillé de récupérer le projet support pour pouvoir démarrer l'application et voir un peu se comment tout cela fonctionne. Vous pourrez également faire des modifications sur le projet avant de vous lancer dans le votre.

Note Afin que vous puissiez démarrer en même temps le projet AusyPlay et le StarterKit, le port du StarterKit a été positionné sur 9001 (et le port JPDA sur 8001), ce qui implique que pour accéder à l'application vous dévrez utiliser l'adresse http://localhost:9001

Contenu

Ce StarterKit contient les éléments suivants :

La version 1.2.3 de Play!Framework
Dernière version stable du framework au moment de rédaction de ce document. Vous pouvez retrouver toutes les informations et la documentation sur le site officiel de Play!.
Le module CRUD
Ce module permet de générer simplement des pages de visualisation, création, modification, suppression de vos objets model
Le module Secure
Ce module permet créer un page d'identification et de gérer des profiles utilisateurs pour sécuriser l'accès à votre application
Le module Guice
Ce module permet d'utiliser Guice pour gérer l'injection de dépendance dans votre projet
La version 1.5.2 de JQuery
JQuery est une librairie javascript relativement géniale, si vous ne savez pas de quoi il s'agit, rendez-vous sur JQuery
La feuille de style Bootstrap Twitter
Cette feuille de style vous permet de donner simplement et rapidement un peu d'allure à votre application. Vous apprendre à l'utiliser, rendez-vous sur Bootstrap Twitter. Cette page utilise d'ailleurs quelques éléments de cette feuille de style
La classe JPQLHelper
Cette classe vous aidera à réaliser vos requête JPQL dynamique en gérant notamment le cas des paramètres nuls ou vide.
Le support de formation basé sur le projet AusyPlay
Comme des exemples de classes de test unitaires et fonctionnels, des Jobs, des classes de base pour vos models ou vos controlleurs ...

Comme des exemples de classes de test unitaires et fonctionnels, des Jobs, des classes de base pour vos models ou vos controlleurs ...

Objectifs

L'objectif de ce StaterKit est donc double :

  1. Proposer quelques outils supplémentaires pour pouvoir démarrer votre projet encore plus rapidement
  2. Proposer une présentation de Play!Framework ainsi qu'un support de formation basé sur le projet AusyPlay
