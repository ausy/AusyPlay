Les Tests

Play! propose trois types de test : Unitaire, Fonctionnel, Selenium.
Je ne vais pas revenir sur le fonctionnement de ces tests qui sont expliqués dans la documentation de Play! mais je vous suggère juste d'aller jeter un oeil sur les tests fonctionnels et sélenium que nous avons mis en place sur la projet AusyPlay car ils montrent comment gérer le cas des pages accessibles uniquement après identification (protégées par le module Secure).

De plus vous pourrez y voir un exemple de test sélénium ainsi que les classes de base de test que nous avons créées et ajoutées au StarterKit.

Lancer les tests : Pour pouvoir éxécuter les tests, taper la commande play test puis pointer sur l'url /@tests
